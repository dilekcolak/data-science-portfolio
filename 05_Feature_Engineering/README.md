# Feature Engineering Projects

This repository contains practical implementations of **feature engineering techniques used in machine learning pipelines**.

Feature engineering is the process of transforming raw data into meaningful input variables that improve the performance of machine learning models. It plays a critical role in data preprocessing and directly affects model accuracy and interpretability.
